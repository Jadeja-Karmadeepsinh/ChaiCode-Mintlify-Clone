### Mintlify Clone

A front-end clone of the Mintlify documentation site built using HTML, CSS, and static assets. This project replicates the look & feel of the original Mintlify landing/docs UI as a practice UI/UX clone.

### Live demo: https://mintlify-clone-wine.vercel.app/

Project Structure
Mintlify-Clone  
├── assets             
├── index.html        
├── style.css          
└── README.md          

### How to Run Locally

Clone the repo

git clone https://github.com/Jadeja-Karmadeepsinh/ChaiCode-Mintlify-Clone.git


Open `index.html` in your browser
